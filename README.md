### css中的[]
- span[class='test']

=>匹配所有带有class类名test的span标签

- span[class *='test']

=>匹配所有包含了test字符串的class类名的span标签

- span[class]

=>匹配所有带有class属性的span标签

- [class='all']

=>匹配页面上所有带有class='all'的标签

- [class *='as']

=>匹配页面上所有class类且类名带有as字符串的类的标签
